STEP 1 — Generate today’s call list

npx tsx server/run-dm-coverage.ts --top=<top> --limit=<limit> --runOpportunity=false

STEP 3 — Process call outcomes
Run call engine:
npx tsx server/run-call-engine.ts

STEP 4 — Freshness guardrail + query intelligence (conditional)
Compute Fresh pool count (use the same logic as query-intel).
If fresh_count < --targetFresh:
  run query-intel:
  npx tsx server/run-query-intel.ts --generate=<generate> --targetFresh=<targetFresh> --market=<market>

  skip query-intel and print “Freshness OK: <fresh_count>”
