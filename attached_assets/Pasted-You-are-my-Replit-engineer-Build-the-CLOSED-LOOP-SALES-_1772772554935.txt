PART 2 — STRUCTURED SCRIPT PATCH TYPES
==================================================

For each patch type define exactly how it influences script generation:

1) add_gatekeeper_redirect_line
Adds a direct authority-seeking line after brush-off.

2) strengthen_value_prop
Makes the operator explain value faster and more concretely.

3) simplify_opener
Shortens the opener if long-winded behavior is detected.

4) add_qualifying_question
Injects one required qualifying question early.

5) add_objection_handler
Adds a rebuttal tied to frequent objection pattern.

6) shorten_response_after_deflection
Forces a tighter follow-up response after gatekeeper stall.

7) reposition_authority_redirect
Moves the redirect earlier in the call flow.

8) add_followup_email_angle
Changes email copy based on objections/outcomes.

9) escalate_to_decision_maker_role_request
If no named DM exists, asks for role more precisely:
Safety Manager / Site Superintendent / Project Manager etc.
